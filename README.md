# Leetcode_CPP

本仓库是我个人的 C++ 刷题记录，按题目来源和练习类型组织，方便复习和查找。

## 当前目录结构

- `剑指Offer/`
  - `剑指Offer-I/`：《剑指 Offer I》题目练习，主要涵盖数组、链表、树、动态规划、字符串等常见算法题。
  - `剑指Offer-II/`：《剑指 Offer II》题目练习，包含进阶题目与变种题型。
- `leetcode/`
  - `leetcode 每日一题/`：每日一题解法记录。
  - `leetcode算法学习/`：算法与数据结构专题训练，例如数组、字符串、二叉树、动态规划等。
- `牛客/`
  - `华为od机试2022/`：华为在线编程机试题目合集。
  - `常见输入输出练习/`：输入输出基础练习，如字符串排序、A+B 等。
  - `牛客华为真题/`：牛客网与华为真题练习题目。
- `test/`：测试代码或临时练习文件。

## 使用说明

1. 打开项目根目录。
2. 进入对应分类目录查找题目文件。
3. 题目文件名一般包含题目编号或题目名称，方便定位。

## 代码风格

- 主要使用 C++ 语言。
- 文件名采用中文题目描述，便于记忆和查找。

## 目标

- 持续补齐算法题，记录思路与代码实现。
- 通过分类整理提高刷题效率。
- 方便以后复习时快速定位问题类型。
